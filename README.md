这是一个用于协助《饥荒：联机版》模组开发的 SKill。

## 环境要求

- 《饥荒：联机版》
- 《Don't Starve Mod Tool》，如果未安装可以告知 AI 让它帮你安装，此操作会拉起 Steam 客户端。
- Python 3
- Codex / Claude Code / OpenCode 等支持 Skill 的 AI 工具

常见游戏目录：

- Windows: `C:\Program Files (x86)\Steam\steamapps\common\Don't Starve Together`
- Linux: `~/.local/share/Steam/steamapps/common/Don't Starve Together`
- macOS: `~/Library/Application Support/Steam/steamapps/common/Don't Starve Together`

常见数据包位置：

- 脚本包：`data/databundles/scripts.zip`
- 图像包：`data/databundles/images.zip`

如果你的游戏不在默认目录，在激活 Skill 时将路径告知 AI。
